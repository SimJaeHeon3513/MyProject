## 기존 코드는 학습 속도가 너무 느린 문제를 발견

### 결측치만 제거한 데이터를 조금 더 전처리를 위해 정규표현식(Regular Expression)을 사용하여 한글과 공백을 모두 제거
```python
import re

def clean_text(text):
    # 한글과 공백만 남기고 특수문자, 숫자 등 제거
    text = re.sub(r'[^가-힣ㄱ-ㅎㅏ-ㅣ\s]', '', text)
    # 중복된 공백 제거
    text = re.sub(r'\s+', ' ', text).strip()
    return text

def preprocess_function(examples):
    # 'document' 컬럼에 clean_text 적용
    examples['document'] = [clean_text(doc) for doc in examples['document']]
    return examples
```
